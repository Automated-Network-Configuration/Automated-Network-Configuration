How to Use the FIU Capstone Switch Configuration Script

This Python script automatically configures a Cisco switch over the serial console using pyserial. It:

  • Detects the correct COM/USB serial port
  • Logs into the switch console
  • Implements VLAN, voice, Quality of Service (QoS), and port configurations
  • Configures the hostname, management IP address, and message of the day (MOTD) banner
  • Saves the running configuration

2. Overview of the Script

The script uses these key modules:

import serial
import serial.tools.list_ports
import time
import re
  • serial & serial.tools.list_ports: Communicate with the switch over the COM port and automatically detect the appropriate port.
  • time: Adds small delays so the switch has time to respond.
  • re: Identifies prompts such as [confirm] or copy confirmation messages in the switch output.

Core Functions

1. detect_com_port()

Identifies the first serial port that matches a USB or serial adapter and prints:

[‚úì] Using detected port: COM3

2. send_and_confirm(ser, cmd, delay=0.4)

Transmits a command, waits for a response, and then:

  • Automatically answers [confirm] prompts
  • Confirms Destination filename for copy run start
  • Prints which command is being sent

3. configure_switch(port)

Connects to the switch on the specified port and runs all configuration commands:

  • Basic security (passwords, local admin user)
  • Console and VTY protection
  • Default gateway and management VLAN SVI
  • Creation of VLANs 10/20/30/40 and port assignments
  • Voice VLAN and QoS configuration
  • Trunk uplink to the UniFi router
  • Shutting down unused port Fa0/48
  • MOTD banner configuration
  • Saving the configuration

4. main()

  • Detects the COM port
  • Calls configure_switch(port)
  • Handles errors and waits for Enter before closing

5. Script Customization

Edit the commands list inside configure_switch() to adapt the configuration.

5.1 Change Hostname

"hostname SW1",

5.2 Change Management IP / Gateway

"ip default-gateway 192.168.1.1",
"interface vlan1",
"ip address 192.168.1.5 255.255.255.0",

5.3 Change VLANs or Port Ranges

Adjust VLAN IDs, names, or interface ranges as needed:

"vlan 20", "name USERS", "exit",
"interface range fa0/17 - 30",
"switchport access vlan 20",

5.4 Change Trunk Port or Allowed VLANs

"interface fa0/47",
"switchport trunk allowed vlan 1,10,20,30,40",

After modifying any commands, re-run the script to push the updated configuration.

6. Error Handling Procedures

  • Waits for keyboard input so you can read the error before the window closes.
